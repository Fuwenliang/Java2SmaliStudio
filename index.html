Java2SmaliStudio

Java Code 2 Smali Plugin for IntelliJ IDEA & Android Studio to easily compile a Java file to smali.

Usage

  1. create or open a Java source file with your desired functionality
  2. click 'Build' and 'Compile To Smali'
  3. wait while IDEA's indicating the compilation process
  4. the Smali file is opened automatically in your editor

Code2Smali is available at JetBrains Plugin Repository and can therefore be found in your IDE's plugin manager! Alternatively, you can download the plugin from the release section or download from Github Repository.
